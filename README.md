# NinjaRobotV4

This project is a reconstruction of the NinjaRobotV3, aiming to improve its structure, maintainability, and reusability. 

## Current Status

The `pi0buzzer` library has been created and is ready for testing. This library provides control over a passive buzzer connected to a Raspberry Pi. Step-by-step testing instructions have been generated and are available.

## Previous Status

- The `ninja_utils` library was created, providing shared utilities for logging and non-blocking keyboard input.
