# NinjaRobotV4

This project is a reconstruction of the NinjaRobotV3, aiming to improve its structure, maintainability, and reusability. 

## Current Status

The `pi0disp` library for the ST7789V display has been created, tested, and documented. It includes bundled fonts, and a global assets directory has been established for easier resource management.

## Previous Status

- The `pi0vl53l0x` library for the VL53L0X distance sensor has been created and is ready for testing.
- The `pi0buzzer` library was created and is ready for testing.
- The `ninja_utils` library was created, providing shared utilities for logging and non-blocking keyboard input.
